# FileHarvest - Chrome Web Store Listing

## Short Description (132 characters max)
Scan webpages and download files in bulk. Features filtering, real file sizes, and AI-powered scanning.

## Detailed Description

**FileHarvest** is a powerful Chrome extension that helps you quickly find and download files from any webpage.

### üöÄ Key Features

**Fast Standard Scanning**
- Instantly detect all downloadable files on any webpage
- Free and unlimited scans
- Works on most websites without configuration

**AI Deep Scan (Optional)**
- Use Google's Gemini AI for advanced file detection
- Perfect for complex pages with dynamic content
- Requires free Google AI API key (get yours at https://aistudio.google.com/apikey)

**Smart File Management**
- Filter by type: Images, Videos, Audio, Documents, Archives
- Real-time search through detected files
- See actual file sizes before downloading
- Select individual files or download multiple at once

**Modern Interface**
- Clean, dark-themed design
- Auto-detects current page URL
- Easy-to-use filtering and search
- Pagination for large file lists

### üìÅ Supported File Types

- **Images**: JPG, PNG, GIF, WebP, SVG, BMP, ICO
- **Videos**: MP4, WebM, AVI, MOV, MKV, FLV
- **Audio**: MP3, WAV, OGG, M4A, FLAC, AAC
- **Documents**: PDF, DOC, DOCX, XLS, XLSX, PPT, TXT
- **Archives**: ZIP, RAR, 7Z, TAR, GZ
- And many more...

### üîí Privacy & Security

- Your API key is stored locally in your browser only
- No data collection or tracking
- Open source and transparent
- Files download directly from original sources

### üí° How to Use

1. **Install the extension**
2. **Click the FileHarvest icon** in your browser toolbar
3. **Choose scan type**:
   - Standard Scan: Quick, free, no setup needed
   - AI Deep Scan: Advanced detection (requires API key)
4. **Filter and select** the files you want
5. **Download** with one click!

### üÜì Completely Free

FileHarvest is completely free to use. The AI Deep Scan feature requires a Google AI API key, which you can get for free from Google AI Studio. Google offers a generous free tier that's sufficient for personal use.

### üìù Permissions Explained

- **downloads**: To download files you select
- **storage**: To securely store your API key locally
- **activeTab**: To scan the current webpage
- **tabs**: To auto-fill current page URL
- **host_permissions (all_urls)**: To scan any webpage you visit
